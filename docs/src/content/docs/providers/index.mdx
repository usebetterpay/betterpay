## Payment Methods

### Virtual Account (VA)
Direct bank transfer through virtual account numbers. Supported banks:
- BCA, BNI, BRI, Mandiri
- CIMB Niaga, Permata, Danamon
- BSI, OCBC NISP, Maybank
- And more...

### E-Wallet
Popular Indonesian e-wallets:
- OVO
- DANA
- ShopeePay
- LinkAja

### QRIS
Universal QR code payment standard:
- All QRIS-enabled apps
- Cross-bank compatibility
- Instant confirmation

### Retail
Cash payment at retail outlets:
- Alfamart
- Indomaret
- Alfamidi

### Credit Card
Visa, Mastercard, and JCB support (provider-dependent)

## Provider Interface

All providers implement the same interface:

```typescript
interface PaymentProvider {
  readonly id: string;
  readonly name: string;
  readonly paymentMethods: PaymentMethod[];
  readonly capabilities: ProviderCapabilities;

  createPaymentLink(data: CreatePaymentLinkInput): Promise<PaymentLinkResult>;
  verifyWebhook(data: WebhookData): Promise<boolean>;
  normalizeWebhook(data: WebhookData): Promise<NormalizedWebhookEvent[]>;
  getApiEndpoint(): string;
  
  checkStatus?(providerTransactionId: string): Promise<StatusResult>;
  cancelTransaction?(providerTransactionId: string): Promise<void>;
  refund?(data: RefundInput): Promise<RefundResult>;
}
```

## Multi-Provider Setup

BetterPay supports multiple providers simultaneously:

```typescript
import { betterPay } from '@betterpay/core';
import { tripay } from '@betterpay/tripay';
import { midtrans } from '@betterpay/midtrans';

const pay = betterPay({
  plugins: [
    tripay({
      apiKey: process.env.TRIPAY_API_KEY!,
      privateKey: process.env.TRIPAY_PRIVATE_KEY!,
      merchantCode: process.env.TRIPAY_MERCHANT_CODE!,
      isSandbox: process.env.NODE_ENV !== 'production',
    }),
    midtrans({
      serverKey: process.env.MIDTRANS_SERVER_KEY!,
      clientKey: process.env.MIDTRANS_CLIENT_KEY!,
      isSandbox: process.env.NODE_ENV !== 'production',
    }),
  ],
});
```

## Choosing a Provider

### For Small to Medium Business
- **Tripay** - Affordable fees, easy integration, good documentation
- **Duitku** - Low fees, suitable for SMEs

### For Enterprise
- **Midtrans** - Comprehensive features, enterprise support
- **Xendit** - Regional coverage, advanced features
- **DOKU** - Enterprise-grade, advanced fraud detection

### For Specific Use Cases
- **QRIS-only**: Any provider with QRIS support
- **Subscription**: Mayar (coming soon)
- **Multi-country**: Xendit (Indonesia, Philippines, Vietnam)
